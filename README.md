# EuclidOS 📐

Welcome to the official web repository for **EuclidOS**. This repository hosts the landing page and documentation.

## About EuclidOS
EuclidOS is an AOSP-based custom ROM focused on:
*   **Precision:** Pixel-perfect UI elements.
*   **Geometry:** Design language inspired by mathematical aesthetics.
*   **Performance:** Stripped-back code for maximum speed.

**Before start flashing.....**
- Always have backup of your internal storage
- Make sure firmware (fw) is flashed if needed

----

**Clean flash:**
- Download the ROM
- Boot into recovery
- Format Data (type yes)
- Go to ADB Sideload
- Flash the ROM via adb sideload
- Reboot to system

----

**Dirty flash:**
- Download the ROM
- Boot into recovery
- Flash the ROM via adb sideload
- Reboot to system
